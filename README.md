# Car Market Analysis & Insights 🏎️📊

## Project Overview
This repository contains a detailed analysis of the automotive market, focusing on luxury, sports, and high-performance vehicles. The goal is to understand the relationship between technical specifications (Engine size, Horsepower, Acceleration) and market pricing using data cleaning and statistical visualization techniques.

## Project Workflow
The analysis is divided into four main stages, represented by the following files:

1.  **Raw Data (`Data Before Cleaning`):** The initial dataset containing various car makes, models, and technical specs with some inconsistencies and missing values.
2.  **Data Cleaning (`Data After Cleaning`):** The refined dataset where data was standardized, missing values handled, and features like 'Torque' were reviewed for consistency.
3.  **Statistical Measures (`Measures`):** Calculated metrics including:
    * Sum of prices per brand.
    * Count of car models per manufacturer.
    * Total horsepower distribution.
    * Market share by production year (2020-2022).
4.  **Final Dashboard (`DashBoard`):** A summary of Key Performance Indicators (KPIs) used to visualize the final findings of the analysis.

## Key Features Analyzed
- **Performance Metrics:** 0-60 MPH times and Horsepower.
- **Market Trends:** Distribution of models across years (2021/2022 dominance).
- **Brand Analysis:** Insights into top manufacturers like Porsche, McLaren, Audi, and Ferrari.
- **Pricing Strategy:** Identifying high-value outliers and average market pricing for different engine types (Patrol, Electric, Hybrid).

## Tools Used
- **Microsoft Excel / Power BI:** For data processing and dashboarding.
- **Pivot Tables & DAX:** For generating statistical measures.

## How to Explore
- Check `Data After Cleaning.csv` for the final structured data.
- Refer to `Measures.csv` for the numerical breakdowns and aggregations.
